Mars V.S. Earth

Project Overview

Goal: Create a web app that fetches weather data (Earth + Mars) using Weather and NASA APIs, then tells users if they are eligible or prepared to "move" to Mars based on environmental comparisons and other conditions.


Programs and Technologies Used

{ Pug } {JavaScript} {Lottie} {Node JS}


APIs Used

• OpenWeatherMap API – For current Earth weather data

• InSight Mars Weather API – For latest Mars weather data
